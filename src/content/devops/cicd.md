# CI/CD Pipelines

Continuous Integration (CI) and Continuous Deployment (CD) form the backbone of modern DevOps practices.

## Continuous Integration
Developers frequently merge their code changes into a central repository, after which automated builds and tests are run.

## Continuous Deployment
Changes that pass all stages of your production pipeline are released to your customers automatically.

## Common Tools
- GitHub Actions
- GitLab CI/CD
- Jenkins
- Azure DevOps
