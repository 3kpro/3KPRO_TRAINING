# Core PowerShell and Bash Commands

Both PowerShell and Bash are powerful command-line interfaces for managing operating systems.

## Bash Basics
- `ls`: List directory contents.
- `cd`: Change directory.
- `cp` / `mv` / `rm`: Copy, move, remove files.
- `cat`: Output file contents to standard output.

## PowerShell Basics
PowerShell uses a Verb-Noun naming convention.
- `Get-ChildItem` (alias `ls` or `dir`): List directory contents.
- `Set-Location` (alias `cd`): Change directory.
- `Copy-Item` / `Move-Item` / `Remove-Item`: Copy, move, remove files.
- `Get-Content` (alias `cat`): Output file contents.
