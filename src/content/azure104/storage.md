# Storage Accounts

An Azure storage account contains all of your Azure Storage data objects, providing a unique namespace for your data accessible from anywhere.

## Storage Services

- **Blob Storage:** Massively scalable object store for text and binary data (like images, documents, and backups).
- **File Storage:** Managed file shares for cloud or on-premises deployments via SMB protocol.
- **Queue Storage:** A messaging store for reliable messaging between application components.
- **Table Storage:** A NoSQL key/value store for rapid development using massive semi-structured datasets.

## Access Tiers
- **Hot:** Optimized for frequently accessed data.
- **Cool:** Optimized for infrequently accessed data (stored for at least 30 days).
- **Archive:** Optimized for rarely accessed data (stored for at least 180 days with flexible latency requirements).
