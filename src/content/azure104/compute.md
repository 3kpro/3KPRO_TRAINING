# Compute Resources

Azure offers various ways to run your applications, from raw virtual machines to fully managed serverless code.

## Virtual Machines (VMs)
Infrastructure as a Service (IaaS). You have full control over the OS and software, but you are responsible for patching and maintenance.

## App Services
Platform as a Service (PaaS) for hosting web applications, REST APIs, and mobile backends. You don't manage the underlying servers.

## Azure Kubernetes Service (AKS)
Managed Kubernetes service for deploying and managing containerized applications at scale.
